Number System Converter

This is an application that enable you to enter a Number system and convert it to another Number System i.e. from or to Decimal, Binary, Octal and Hexadecimal.

How it works is This:

  • Enter a Number.

  • Select the type of Number System you entered.

  • Then, click the Button corresponding to the Number System you want to covert to.

  • The result will then be displayed for you below the result label.

Multiplication Table Generator

This is a project that enable you to generate Muliplication table of any Number which you enter into the Number input field on the UI from 1 to any Number you enter into the range input field provided in the application.

How it works is This:

  • Enter a Number you want to generate its Multiplication table.

  • Enter the range of where you want the Table to stop into the range input field provided.

  • Then, click the generate Table Button

  • The result will then be displayed below for you.

  • You can also clear the input field so as to enter new Number and range. You can as well update either of the input field, click the generate Button, and the new updated result will be generated for you.
